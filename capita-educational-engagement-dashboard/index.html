CAPITA Educational Engagement Dashboard


The core aim of our project is to produce a system that automatically gathers and processes engagement information from social media platforms about schools and colleges across the UK. This helps education institutions improve the education they provide.

Our solution is a powerful API that automatically gathers topic-related information from various sources: Twitter, Facebook, and The Student Room. The data extracted is associated with a UK Educational Institution and it is processed so that actionable insights are provided in a variety of accessible formats.

Once our system has collected the data, it runs various algorithms to produce relevant insights and discoveries (trends and patterns) that will benefit our client.

Key Features

The main features of the system are...

Topic-Related Data Capturing

Our system captures specific information from various sources including Facebook, Twitter and The Student Room Forum.

Interactive Dashboard

Our system should not just be used by future developers. Every user can get the desired insight over any educational institution by using our interactive interface.

Accurate Analysis

Powerful analysis algorithms, such as Named-Entity Recogniser, Natural Language Processing and Topic Modeling, are runned over the extracted data for an improved accuracy of the information provided.

Data Storage

As our system is designed to collect big amounts of data, we are using PostgreSQL: the most advanced open-source database, optimised for complex data models.
